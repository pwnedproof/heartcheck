# 💔 HeartCheck

A simple tool that gives you an honest answer about your situation. Training an LLM to give better responses

🌹 https://pwnedproof.github.io/heartcheck/

---

## What is this?

HeartCheck takes whatever you type about someone you're dealing with and gives you a straightforward answer:

- YES — it’s going somewhere  
- WAIT — mixed signals  
- NO — not worth it  
- MOVE ON — you already know  

No accounts, no data, no overthinking. Just type and get a response.

---

## How it works

Under the hood, it looks for patterns in what you write.

It checks for things like:
- effort (making plans, consistency)
- lack of effort (ghosting, flaking, only texting late)
- mixed signals (hot and cold, confusing behavior)

Each of these adds or subtracts from a score.  
That score decides the final result.

It also calculates things like:
- how confident the result is  
- how risky the situation looks  
- whether you're being a little delusional (respectfully)  
- who’s putting in more effort  

The same input will always give the same result, but the wording changes so it doesn’t feel repetitive.

---

## Why I made this

Sometimes you don’t need advice — you just need clarity.

This is basically the honest friend that tells you what you already know but might be avoiding.
